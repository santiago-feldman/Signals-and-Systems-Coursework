# Signals and Systems Coursework | ITBA

This repository contains the practical assignments done for the following subjects at Instituto Tecnológico de Buenos Aires:

- 22.05 Digital Signals and Systems
- 22.67 Random Signals

## Digital Signals and Systems

- **TP0 - Python signal sampling simulation**
- **TP1 - Sampling** (analog sampling and recovery PCB with an antialiasing filter, an analog switch, and a Sample & Hold)
- **TP2 - FFT + MIDI Sythesizer** (Cooley-Tukey algorithm to implement the Fast Fourier Transform in C + MIDI synthesizer with FM, Karplus-Strong and sample-based synthesis, with digital audio effects)
- **TP3 - Proteous and Python simulation of ADC/DAC architectures** (DAC0800/ADC0808, sigma-delta with noise shaping, delta modulation, SAR)
- **TP4 - Comprehensive analysis of the YASA (Yet Another Spindle Algorithm) library for sleep characterization** (paper)
- **TP5 - FIR (Kaiser-window) and IIR filters implemented and measured on the ESP32 CPU**
- **TPF - Adaptive filtering for Active Noise Control (ANC)** (from Wiener optimal filtering and stochastic-gradient descent / SGD to Filtered-x LMS/RLS (FxLMS/FxRLS)**
